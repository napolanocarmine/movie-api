# nestjs-movie-api
In this challenge, I have to create a Movie REST API using NestJS, a typescript framework for building efficient and scalable Node.js applications. I will be implementing CRUD operations for movies and related genres.


## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
